# 数据标注工具 - 启动与打包指南

## 项目概述

这是一个基于 **Electron + TypeScript + React** 的桌面数据标注工具，支持 Excel 上传、BERT 模型推理、结果反馈收集。

### 技术栈

| 层级       | 技术                        |
| -------- | ------------------------- |
| 桌面框架     | Electron 29 + TypeScript  |
| 前端 UI    | React 18 + Tailwind CSS   |
| 数据存储     | better-sqlite3（本地 SQLite） |
| Excel 解析 | xlsx (SheetJS)            |
| 推理服务     | Python + Flask + BERT     |
| 打包工具     | electron-builder          |

***

## 环境要求

### 必需环境

1. **Node.js** >= 18.0.0

   * 下载: <https://nodejs.org/>

   * 验证: `node -v`

2. **Python** 3.8 - 3.11（推荐 3.10）

   * 下载: <https://www.python.org/downloads/>

   * 验证: `python --version` (Windows)

   * **重要**: 安装时勾选 "Add Python to PATH"

   * **注意**: Python 3.12 暂不支持 PyTorch 2.0，请勿使用

3. **npm** >= 9.0.0（随 Node.js 安装）

   * 验证: `npm -v`

## 启动程序

### 方法一：使用快速启动脚本（推荐）

项目根目录提供了 `start.bat` 脚本，会自动检查环境并启动：

```bash
# 进入项目目录
cd e:\laboratory\AI\Fine-tuning\data-label-tool

# 运行启动脚本
start.bat
```

脚本会执行以下操作：

1. 检查 Node.js 是否安装
2. 检查 Python 是否安装（警告但不阻止）
3. 检查并安装 npm 依赖（如果 node\_modules 不存在）
4. 启动开发服务器

### 方法二：手动启动

#### 1. 安装前端依赖

```bash
cd e:\laboratory\AI\Fine-tuning\data-label-tool
npm install
```

#### 2. 安装 Python 依赖

```bash
cd python_service
pip install -r requirements.txt
cd ..
```

Python 依赖包括：

* flask>=2.3.0

* flask-cors>=4.0.0

* torch>=2.0.0

* transformers>=4.30.0

* numpy>=1.24.0

#### 3. 开发模式运行

```bash
npm run dev
```

这将同时启动：

* Vite 开发服务器（前端热更新）- <http://localhost:5173>

* Electron 窗口（桌面应用）

* Python 推理服务（自动启动，端口 5000）

## 打包生成桌面程序

### Windows 安装程序（.exe）

#### 方法一：使用打包脚本（推荐）

```bash
# 进入项目目录
cd e:\laboratory\AI\Fine-tuning\data-label-tool

# 运行打包脚本
build.bat
```

脚本执行流程：

1. 检查 Node.js 环境
2. 安装 npm 依赖（如果缺失）
3. 安装 Python 依赖
4. 构建前端应用（TypeScript 编译 + Vite 构建）
5. 打包 Windows 安装程序（使用 electron-builder）

#### 方法二：手动打包

```bash
# 1. 安装依赖
npm install

# 2. 构建前端
tsc && vite build

# 3. 打包 Windows 安装程序
npm run build:win
```

### 打包输出

打包完成后，安装程序位于：

```
release/数据标注工具 Setup 1.0.0.exe
```

### 打包配置说明

`package.json` 中的 `build` 字段配置：

* **应用 ID**: `com.yourcompany.data-label-tool`

* **产品名称**: `数据标注工具`

* **安装程序**: NSIS（支持自定义安装路径）

* **快捷方式**: 桌面 + 开始菜单

* **架构**: x64

* **图标**: `build/icon.ico`

### 其他平台打包

```bash
# macOS (DMG)
npm run build:mac

# Linux (AppImage)
npm run build:linux
```

## 安装程序使用

### 安装过程

1. 运行 `release/数据标注工具 Setup 1.0.0.exe`
2. 选择安装语言（中文）
3. 接受许可协议

7. 点击"完成"启动应用

* 或控制面板 → 程序和功能 → 卸载

## 常见问题

### Q1: 启动时提示 "Python 服务未启动"

**原因**: Python 环境未配置或依赖未安装

**解决**:

```bash
# 检查 Python
python --version

# 安装依赖
cd python_service
pip install -r requirements.txt
```

### Q2: 打包后 Python 服务无法启动

**原因**: 生产环境 Python 路径问题

**解决**: 确保目标机器已安装 Python，或在打包时将 Python 环境一并打包（使用 PyInstaller）。

### Q3: 端口冲突

**解决**:

* Vite 开发服务器默认使用 5173 端口

* Python 服务默认使用 5000 端口

* 如果端口被占用，检查并关闭占用端口的程序
